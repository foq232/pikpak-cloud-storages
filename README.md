## TeraBox, 왜 갈아타려는 걸까?

솔직히 TeraBox가 완전히 나쁜 서비스라고 하기는 어렵다. 1TB 무료 공간은 실제로 매력적이고, UI도 비교적 깔끔하다. 그런데 몇 가지 불편한 진실이 있다.

**첫째, 태생이 바이두 계열이다.** TeraBox의 전신은 'Dubox'인데, 바이두 재팬의 자회사인 Flextech Inc.에 인수된 서비스다. 서버는 일본으로 옮겼다지만, 사실상 중국 기업의 연결고리가 있다 보니 민감한 파일을 올리기가 꺼려진다는 의견이 많다.

**둘째, 무료 플랜의 다운로드 속도가 심각하게 느리다.** 업로드는 그나마 빠른데, 내려받을 때는 속도 제한이 걸린다. 프리미엄 없이는 제대로 된 사용이 어렵다는 현실적인 문제.

**셋째, 토렌트 오프라인 다운로드 성능이 미지근하다.** TeraBox도 '원격 업로드' 기능을 제공하긴 하는데, 실제 사용자들 후기를 보면 "근성이 약해서 별로 소용없다"는 평가가 꽤 많다. 토렌트 파일 저장이 주목적이라면 결정적인 단점이다.

이런 이유로 **Terabox 대체**를 찾아다니는 사람들이 자연스럽게 PikPak에 눈길을 돌리게 된다.

## PikPak이 뭔데?

PikPak은 싱가포르 기업 PIKCLOUD PTE. LTD.가 2021년에 출시한 프라이빗 클라우드 서비스다. 한국에서는 주로 **토렌트 클라우드** 용도로 알려졌는데, 기능 자체는 그보다 훨씬 폭넓다.

핵심을 한 줄로 요약하면: **마그넷/토렌트 링크를 픽팍 서버가 대신 받아두고, 나는 그냥 스트리밍하거나 다운로드만 하면 된다.**

내 컴퓨터가 켜져 있을 필요도 없고, 내 IP가 토렌트 네트워크에 노출될 필요도 없다. 픽팍 서버가 다 해주는 구조다.

## PikPak의 핵심 기능

### 오프라인 다운로드 (클라우드 다운로드)

픽팍의 가장 강력한 기능이자 TeraBox 대비 확실히 앞서는 부분이다. 지원하는 형식이 꽤 넓다.

- 토렌트 (.torrent 파일) 및 마그넷 링크
- ED2K 링크
- HTTP/HTTPS 직링크
- X(구 트위터), 틱톡, 페이스북 등 소셜 미디어 영상 링크

픽팍 서버에 이미 해당 파일이 있으면 거의 즉시 저장이 완료되는 경우도 많다. 500GB 이상의 파일도 수초 안에 클라우드에 저장이 완료됐다는 후기가 실제로 있다. 물론 완전히 신규 파일이면 시간이 걸리지만, 흔히 공유되는 파일들은 이미 캐싱되어 있는 경우가 많다.

### 원본 화질 4K 온라인 스트리밍

저장된 영상을 클라우드에서 직접 스트리밍할 수 있다. 유튜브처럼 재생하면 되는데, 4K 원본 화질까지 지원한다. 일반 온라인 영상 플랫폼이 압축된 화질로 제공하는 것과 달리, 픽팍은 원본 그대로 재생이 가능하다.

### 텔레그램 봇 연동

`@PikPak_Bot`에 마그넷 링크나 파일을 전송하면 픽팍 클라우드에 자동 저장된다. 텔레그램을 주로 쓰는 사람이라면 굉장히 편리한 워크플로우다.

### 브라우저 확장 프로그램

Chrome, Edge 확장 프로그램을 설치하면 웹 페이지에서 토렌트/마그넷 링크를 자동 감지해 원클릭으로 픽팍에 저장할 수 있다.

### WebDAV 지원

rclone 같은 도구를 이용하면 픽팍을 윈도우 탐색기에 가상 드라이브로 마운트할 수 있다. NAS를 쓰는 사람들이 픽팍을 외부 클라우드로 연결해서 쓰는 방식도 가능하다.

### 멀티 플랫폼

Android, iOS, Windows, macOS(출시), 웹 버전, TV 앱까지 지원한다. 어디서든 접근 가능하다는 것이 기본 중의 기본이긴 한데, 이건 제대로 갖춰져 있다.
